iBeans Channels Examples

The examples below are code snippets that show how do do various tasks with iBeans such as send and receive JMS or email messages, Use the @Schedule, @Send, @Receive and @ReceiveAndReply annotations. As well as examples of how to use method parameter annotations such as @ReceivedHeaders, @SendHeaders and @MessagePayload. This are all Unit tested with the iBeans continuous build so are always up to date and working.
Name Description Demonstrates
pub-sub (unit tests)A collection of example code that shows how a server side component can publish and subscribe messages.Component method annotations: @Schedule, @Send, @Receive and @ReceiveAndReply
params (unit tests)Demonstrates how to use method parameter annotations.Parameter annotations: @ReceivedHeaders, @SendHeaders, @MessagePayload
MEPs (unit tests)Message Exchange Patterns define common ways of exchanging data between componentsPatterns: InOnly, OutOnly, InOut, InOnly-OutOnly
request-response (unit tests)How to define a simple request-response component with auto transformation to XML documentThe @ReceiveAndReply and @MessagePayload annotations
async-reply (unit tests)How to use the @Reply annotation to define an async callback channelThe @Reply, @ReceiveAndReply and @Send annotations
splitter (unit tests)How to split outbound messages into multiple partsThe @Send annotation and its 'split' attribute. Shows how to split an XML document
email-channels (unit tests)Examples of how to receive and send email.The @Send and @Receive annotations and using the Gmail iBean
jms-channels (unit tests)Examples of how to receive and send Jms Messages.The @Send and @Receive annotations and using Guice to configure a Jms Channel.
