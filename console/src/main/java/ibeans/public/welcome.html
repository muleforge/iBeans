WebApp Integration Just Got Easier

Mule iBeans is a new platform integrating web applications with the rest of the world. Mule iBeans provides a simple API and annotations for doing common tasks such as sending email, talking REST, sheduling tasks, Jms, RSS/Atom and more.

 

Mule iBeans also provides a revolutionary way of sharing client services, called Integration Beans that provide a unique way to quickly create clients for web APIs such as Twitter, Flickr, FaceBook, Amazon WS, LinkedIn, BaseCamp and any other service on the web that offers a public REST or Web Service API or protocols such as smtp, imap, xmpp, etc.

 

Discover

Use the ibeans store to discover iBeans that you can use in your applications. This is how reuse is supposed to be.

Play

Try out new iBeans right here in the console. You can even use the iBeans shell to create simple applications.

Configure

Taylor your iBeans container to the needs and constraints of your environment by only enabling the plugins you need.
