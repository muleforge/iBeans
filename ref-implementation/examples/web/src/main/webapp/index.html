Examples

These are a selection of examples that demonstrate the iBeans that are made available in the iBeans distribution. Each of the examples use either the JavaScript client or JSPs to provide quick demos on how to use the iBean.

Name Description Demonstrates
Flickr IBean A JavaScript demo that uses the FlickrIbean to search for photos on Flickr and displays the results. JavaScript client, JSON RPC, Flickr, External config (Flickr API Key)
Flickr IBean (from JSP) A JSP demo that uses the FlickrIbean to search for photos on Flickr and displays the results. JavaScript client, JSON RPC, Flickr, External config (Flickr API Key)
Flickr IBean with Email (from JSP) A JSP demo that uses the FlickrIbean to search for photos on Flickr and allows the user to send the photos to their email account using the the Gmail iBean. JavaScript client, JSON RPC, Flickr, Gmail, External config (Flickr API Key, Gmail credentials)
Twitter IBean A JavaScript demo that uses the TwitterIbean to display the public Twitter timeline. Users can also log in to their own twttier account with this example and view their friend timeline and update their status. JavaScript client, JSON RPC, Twitter, External config (Twitter credentials)
